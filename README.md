# Analisis Potensi Pemanenan Air Hujan (RWH) - Aceh Barat

Proyek ini melakukan analisis spasial untuk menghitung potensi ketersediaan air hujan di Aceh Barat.

## 1. Hasil Analisis (Peta Interaktif)
Hasil pemetaan potensi air hujan dapat diakses secara langsung melalui link berikut:
[Klik di sini untuk melihat Peta Interaktif](https://rijaleffendi2002-maker.github.io/aceh-barat-rwh/)

## 2. Struktur Direktori
- `data/`: Folder penyimpanan data (.zip, .xlsx, .csv)
- `script/`: Folder penyimpanan skrip Python (`analisis_rwh_aceh_barat.py`)
- `index.html`: File peta interaktif hasil analisis

## 3. Langkah-Langkah Menjalankan Ulang (Reproduction)
Untuk menjalankan ulang analisis ini di komputer Anda, ikuti langkah berikut:

1. **Download Proyek**: Klik tombol hijau **Code** di bagian atas repositori ini, lalu pilih **Download ZIP**.
2. **Ekstrak File**: Buka file ZIP tersebut dan ekstrak ke folder di komputer Anda.
3. **Persiapan Lingkungan**: Pastikan Anda telah menginstal Python. Instal library yang dibutuhkan dengan perintah berikut di terminal:
   ```bash
   pip install -r requirements.txt

4. **Jalankan Skrip**:
   - Masuk ke folder script/.
   - Jalankan file analisis_rwh_aceh_barat.py menggunakan Python.

5. **Cek Hasil**: Setelah skrip selesai dijalankan, file peta terbaru akan diperbarui di folder utama.
