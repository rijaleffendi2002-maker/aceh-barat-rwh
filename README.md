# Analisis Potensi Pemanenan Air Hujan (RWH) - Aceh Barat

Proyek ini melakukan analisis spasial untuk menghitung potensi ketersediaan air hujan di Aceh Barat.

## Struktur Direktori
- `data/`: Folder data (.zip, .xlsx, .csv)
- `script/`: Folder berisi `analisis_rwh_aceh_barat.py`
- `README.md`: Penjelasan proyek
- `requirements.txt`: Daftar library
- `LICENSE`: Lisensi MIT
- `DATA_LICENSE.txt`: Lisensi data CC-BY 4.0
